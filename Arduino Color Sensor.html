Arduino Color Sensor

Overview

Inspired by the difficulty of matching specific sneaker shades (like Jordan Orange) with online apparel, this project involved creating a handheld color-sensing device. The system uses an Arduino Mini and a color sensor to capture the exact RGB values of any physical object. The goal is to bridge the gap between physical colors and digital shopping, allowing users to find products that are a 100% tonal match rather than relying on inconsistent screen renders.

My Role & What I Did

  • Hardware Prototyping: Built and wired the color-sensing pen using an Arduino Mini and an optical sensor.
  • Environmental Optimization: Solved the "ambient light interference" problem by designing a light-sealed enclosure and integrating a dedicated light source to ensure consistent readings regardless of external conditions.
  • Software & AI Integration: Developed the logic to convert sensor data into digital RGB codes. I used AI to optimize the code, refining the sensor’s accuracy to ensure it produces stable, repeatable color values every time.
  • System Architecture: Initiated the development of a web-based interface intended to take those RGB codes and scrape the internet for matching apparel.

The Outcome

The project successfully moved from concept to a high-accuracy working prototype. Testing confirmed that the device consistently identifies identical RGB values for a specific shade, overcoming the common hurdle of variable lighting. While the search website remains under construction, the core hardware is fully functional and capable of translating physical "real-world" colors into actionable digital data with high precision.
